Dual UR5 Hand Teleoperation System

Real-time hand gesture control for dual robotic arms using MediaPipe and ROS2.

An innovative teleoperation system that allows intuitive control of two UR5 robotic arms through hand gestures captured via webcam. The system uses MediaPipe for hand tracking and ROS2 for robot control, providing a natural and responsive interface for dual-arm manipulation.

ROS2 Humble Python 3.10 MediaPipe OpenCV URDF RViz Computer Vision

Key Features:

  • Hand tracking with 21 landmarks per hand using MediaPipe
  • Real-time control loop at 50Hz with exponential smoothing
  • 3D visualization with detailed robot meshes in RViz
  • Multiple control modes: Hand tracking, Keyboard, Automatic demo
  • Independent dual arm control with mirrored hand mapping
  • Pinch gesture for gripper control
  • Complete documentation and setup guides
